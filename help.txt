Usage: firecrawl [options] [command]

CLI tool for Firecrawl web scraping

Options:
  -V, --version                       output the version number
  -k, --api-key <key>                 Firecrawl API key (or set
                                      FIRECRAWL_API_KEY env var)
  --api-url <url>                     API URL (or set FIRECRAWL_API_URL env var)
  --status                            Show version, auth status, concurrency,
                                      and credits
  -h, --help                          display help for command

Commands:
  scrape [options] [urls...]          Scrape one or more URLs. Multiple URLs are
                                      scraped concurrently and saved to
                                      .firecrawl/
  download [options] <url>            Download a site into .firecrawl/ as nested
                                      directories. Maps the site first to
                                      discover pages, then scrapes them.
  crawl [options] [url-or-job-id]     Crawl a website using Firecrawl
  map [options] [url]                 Map URLs on a website using Firecrawl
  search [options] <query>            Search the web using Firecrawl
  agent [options] <prompt-or-job-id>  Run an AI agent to extract data from the
                                      web
  browser [options] [code]            Launch cloud browser sessions and execute
                                      Python, JavaScript, or bash code remotely
                                      via Playwright
  claude [options] [input...]         AI workflows powered by Claude Code
  codex [options] [input...]          AI workflows powered by Codex (coming
                                      soon)
  opencode [options] [input...]       AI workflows powered by OpenCode (coming
                                      soon)
  config [options]                    Configure Firecrawl (login if not
                                      authenticated)
  view-config                         View current configuration and
                                      authentication status
  login [options]                     Login to Firecrawl (alias for config)
  logout                              Logout and clear stored credentials
  init [options] [template]           Set up Firecrawl: install CLI,
                                      authenticate, add integrations, and
                                      scaffold a template
  setup [options] <subcommand>        Set up individual firecrawl integrations
                                      (skills, mcp)
  env [options]                       Pull FIRECRAWL_API_KEY into a local .env
                                      file
  credit-usage [options]              Get team credit usage information
  version [options]                   Display version information
  help [command]                      display help for command
